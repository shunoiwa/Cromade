# <img width="28" height="28" alt="icon" src="https://github.com/user-attachments/assets/ac4eb3d2-c4d3-4217-ba61-ce778dc44fa9" /> Cromade 


強制遷移を粛清する安心ブラウザ。

## 開発支援について
このソフトウェアはGitHubで無料公開していますが、Buy Me a coffeeによる開発支援を受け付けています。
支援の有無によって私のコーヒー摂取量が変化しますが、ソフトウェアの機能・利用条件・サポート内容に差はありません。

## 特長
ページの進む/戻る、リロードやホーム表示など、確実な安全な場合を除いて、ページ遷移時は必ずあなたに確認をとります。
画像や動画のダウンロードが簡単にできます。
※法的に問題ないコンテンツのみをダウンロードするようにしてください。

## 機能
> バージョン番号x.y.zについて...
> - x ... システム全体の大幅な変更
> - y ... いくつかの機能追加
> - z ... バグや細かい仕様の修正
> を含みます。
> x.y.0, x.y.1, ... の新機能はすべてx.yにまとめて示します。

### v0.1から
- 画像や動画のダウンロード
- 複数の画像を連番リネームして保存
- 複数の画像を適切なサイズにしてPDF化(おおよそページ数×1.2MB以内に収まります)
- タブ切り替え機能
- 言語設定(日本語、英語)
- お気に入り登録
- 検索エンジンの設定
- ホワイトリストとブラックリスト
  - ホワイトリストに登録されたドメインは承認をスキップ
  - ブラックリストに登録されたドメインは即粛清
### v0.2から
- ダークテーマ / ライトテーマ の選択機能
- 閲覧履歴の保存
- プライベートモード
### v0.3から
- ローカルへのアクセス禁止機能
- アニメーション画像や音声のダウンロード
- メディア取得の強度設定
  - 簡易モード：画像のような即取得可能なメディアのみ対象
  - 標準モード：バランスのとれた設定
  - 強力モード：動画等を含むほぼすべてのメディアを取得するが、問題が起きやすい
- 実行ファイルのダウンロード禁止機能
- パスワードマネージャー
### v0.4から
- ページ内検索
  - 通常検索と正規表現検索に対応
  - Ctrl+Fで検索バーを表示
- チュートリアル
  - 初回起動時に日本語チュートリアル / Tutorial in English / スキップを選択可能
  - ボタンやエリアを一つずつハイライトし、吹き出しとステップ数表示で説明
  - ヘルプメニューからいつでも再実行可能
  - チュートリアル用メディアページでスキャンと保存の流れを確認可能
- ズーム設定
  - ブラウザ全体のズームを設定可能
  - メニューから拡大、縮小、リセットも可能
- フォント設定
  - PCに保存されているフォントからブラウザ全体の表示フォントを選択可能
- サイト別設定
  - 設定画面の「個別設定」タブでサイトごとのズーム、フォント、メディア検出モードを設定可能
- リーダーモードとページスクショ
  - ツールバーから本文抽出ビューを一時表示可能
  - 現在ページのスクリーンショットをPNGとして保存可能
- レッドモード
  - プライベートモードより安全寄りの一時隔離タブ
  - タブ名の先頭に[R]を表示し、タブ色は赤で表示
  - Altクリック、またはCtrl+Shiftクリックで新規レッドタブを作成可能
- 権限設定
  - 設定画面の「権限設定」タブで、ノーマル / プライベート / レッドごとに禁止する機能をチェック可能
  - 履歴保存、Cookie保存、パスワード、ローカルアクセス、サードパーティCookie、通知、デバイスアクセスをモード別に禁止可能
- 粛清レポート
  - 「やろうとしたこと」「許可」「粛清」をタイムライン表示
  - 許可されたものは緑、粛清されたものは赤で表示
  - 承認待ち、手動許可、手動拒否、自動ブロック、自動許可の流れを確認可能
  - JSONまたはTXTでエクスポート可能
- 証明書/HTTPエラー表示
  - 証明書エラーとHTTP読み込みエラーの警告表示をセキュリティ設定から切り替え可能
- デバイスアクセス制限
  - カメラ、マイク、位置情報、通知などのデバイスアクセスを承認制または禁止に設定可能
- JavaScript許可設定
  - セキュリティ設定からJavaScriptの有効/無効を切り替え可能
  - BrowserView生成時に決まるため、変更は次回起動から適用
- 危険ドメイン警告と拡張子偽装チェック
  - Punycode、@入りURL、IP直指定、短縮URLなどを危険ドメインとして警告
  - 二重拡張子やMIME不一致のダウンロードを検出して承認カードで確認
- トラッキング防止機能
